Requerimientos generales:
Usted ha sido contratado para realizar la siguiente maqueta web (ver archivos adjuntos) teniendo en cuenta los siguientes requerimientos: Archivo Adjunto

Requerimientos específicos:
1.- Crear los archivos index.html, estilos.css y su correspondiente carpeta de imagenes. Investigar sobre la propiedad css object-fit y aplicarla en las imágenes del portafolio.

2.- Maquetar idénticamente la página web usando CSS GRID y flexbox (Ver archivo adjunto "layout_desafio3.png"). No debe trabajar con floats o position Esta maqueta debe apreciarse bien para pantallas mayores a 1024px.

3.- Establezca media queries para una vista horizontal (landscape) entre 724px y 1024px, puede ordenar las imagenes como prefieran.

4.- Establezca media queries para la vista movil. Puede ordenar los elementos como prefiera. Con solo indicar las media queries es suficiente, pero es libre de maquetar si asi lo desea.

5.- Además, al pasar el mouse sobre alguna de las imágenes del portafolio, éstas deben mostrar un fondo transparente como se muestra en la imagen adjuntada.
